# Visual Architecture and UI

## Structural Layout
- **AppLayout:** The root wrapper that dictates the grid system, responsive sidebar navigation, and main content area spacing.
- **NxTopBar:** A persistent omni-header containing the global search command bar (`NxCommandBar`), job monitor access, and theme switching controls.

## Specialized Visualizations
The UI incorporates advanced data visualizations:
- **NxActivityHeatmap:** Cognitive trace tracking, styled similarly to GitHub commit graphs.
- **NxMemoryMiniGraph:** Horizontal spectrum graphs for memory and token tracking.
- **NxRelationTimeline:** Interactive timelines for displaying connected intelligence events.
