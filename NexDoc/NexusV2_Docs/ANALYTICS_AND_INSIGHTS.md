# Analytics and Insights

## Key Metrics Tracked
- **Model Performance:** Time to First Byte (TTFB), total processing time, and token utilization across different AI models (e.g., Gemini).
- **Agent Efficiency:** Success rates of agent-executed tasks, error rates, and operational status logs.
- **System Health:** Endpoint latency, WebSocket connection stability, and API rate limit occurrences.

## Visualization Tools
- The platform utilizes components like `DashboardChart.tsx` (wrapping libraries like Recharts or Chart.js) to provide time-series analysis of system performance.
- `NxActivityHeatmap` offers a granular view of cognitive trace tracking and event frequency.
