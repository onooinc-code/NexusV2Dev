# User Personas

## 1. The Operations Manager
- **Needs:** High-level overview of system health, active workflows, and agent statuses.
- **Key Hubs:** AIModelsHub, TasksHub, WorkflowHub.

## 2. The AI Engineer
- **Needs:** Deep access to agent persona configurations, MCP server connections, and prompt debugging.
- **Key Hubs:** AgentsHub, SettingsHub, HedraSoulHub.

## 3. The Analyst / Researcher
- **Needs:** Ability to map relationships, synthesize data from past interactions, and retrieve episodic memories.
- **Key Hubs:** ContactHub, HedraSoulHub.
