================================================================================
                      🆘 TROUBLESHOOTING
================================================================================

If Redis is not running:
  $ docker ps | findstr redis
  
  If not showing up:
  $ docker run -d -p 6379:6379 --name redis-nexus redis:latest

If connection fails:
  1. Check Docker is running
  2. Check container is running: docker ps
  3. Test connection: docker exec redis-nexus redis-cli ping
  4. Check firewall isn't blocking port 6379

If queue worker exits:
  1. Verify: QUEUE_CONNECTION=redis in .env
  2. Check Redis is running
  3. Look at logs: docker logs redis-nexus
  4. Check backend logs: storage/logs/

If cache isn't working:
  1. Verify: CACHE_STORE=redis in .env
  2. Clear cache: php artisan cache:clear
  3. Test: php artisan tinker → Redis::ping()
