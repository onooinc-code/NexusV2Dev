# Temporal and Spatial Awareness

## Contextual Injection
The system is designed to provide agents with real-world grounding:
- **Spatial:** Integration with external utilities (like `api.open-meteo.com`) allows location context injection into agent prompts.
- **Temporal:** Timestamps are strictly managed. The `HedraSoulHub` maps episodic memories across time, visualized via `NxRelationTimeline`.
