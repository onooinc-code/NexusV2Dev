# Design System

## Philosophy
The Nexus design philosophy utilizes a "Cosmic Slate" dark-themed glassmorphism approach. It emphasizes tech-focused visualizations.

## Typography
- **Primary:** Inter (sans-serif) for legibility in UI elements.
- **Monospace:** JetBrains Mono for code blocks, logs, and technical data.

## Color Palette
- Deep dark gradients as backgrounds.
- Neon accents, specifically `nexus-blue` and `hedral-purple` for interaction states and highlights.

## Core Components
- **Buttons:** `NxActionButton` with distinct variants.
- **Cards:** `NxGlassCard` providing translucent, blurred background effects.
- **Inputs:** Standardized `NxInput` fields ensuring accessibility and consistency.
